# Screenshots Folder

This folder contains screenshots of the Pratik Browser interface and features.

## Adding Screenshots

1. Take screenshots of the browser showing different features
2. Save them here with descriptive names
3. Update the README.md to reference the new screenshots

## Example Screenshot Names

- `main-interface.png` - Main browser window
- `tabbed-browsing.png` - Multiple tabs open
- `bookmarks-manager.png` - Bookmarks interface
- `history-view.png` - History tracking
- `dark-mode.png` - Dark theme interface
- `liquid-glass-design.png` - Showcase of the glass aesthetic
